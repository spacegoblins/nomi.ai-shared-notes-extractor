# Changelog

## Version 1.5 — March 23, 2026

### What's New

- **Settings panel, right in the page.** There's now a ⚙ button in the Nomi.ai page header, right next to Import and Export. Clicking it opens a Settings panel where you can choose which export formats to include. Note: You can adjust these same settings from clicking the Extension icon in the Firefox toolbar. The settings sync when you close and reopen either panel.

- **About tab.** The same ⚙ panel has an "About" tab that shows the latest release version, release date, and release notes fetched directly from GitHub.

- **Sections auto-expand after import.** When you import a file, any section whose content changed will now automatically open so you can review it.
