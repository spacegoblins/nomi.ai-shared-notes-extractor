# Nomi.AI Shared Notes Extractor

## Overview:

Nomi.AI Shared Notes Extractor is a Firefox browser extension designed to simplify the process of exporting your Nomi's entire Shared Notes to a simple, human readable .txt file. This is useful for planning, archiving, sharing, and more.

Take a look at the code before using it. If you're a software developer, and have a suggestion to improve this tool, send me an email at spacegoblins@protonmail.com. You're also welcome to modify this extension and upload it to the mozilla store under a new name.

## Installation:

1. In Firefox, search [addons.mozilla.org](https://addons.mozilla.org) for `Nomi.AI Shared Notes Extractor`
2. Click the "Add to Firefox" button
3. Make a selection for private windows and click "Add" on the popup
4. Make a selection for pin to toolbar and click "Done"

## Usage:

1. Navigate to your Nomi's private chat OR their Shared Notes page at beta.nomi.ai
2. Click the Extensions button (puzzle icon) in the top right of Firefox
3. Select the extension
4. Click "Export .txt"
5. Check your browser's default downloads folder for the text file

Now that you have a plain text, human readable file of your Nomi's Shared Notes, you can copy and paste it wherever you like - Share your Shared Notes with other users, ask an LLM for help brainstorming character changes, or more.
